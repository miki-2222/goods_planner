## サービス概要
購入予定のアイテムを管理し、計画的にアイテム購入ができるアプリです。

「発売日や予約日がバラバラで管理が大変！」「このグッズどこで予約してたっけ？」
そんな不安やめんどくさい管理を解消します。

購入したいアイテム、購入済みアイテムをまとめて管理するサービス。


## このサービスへの思い・作りたい理由
きっかけは本の購入忘れとBlu-rayの重複購入でした。
私はグッズ収集が好きなのですが、発売日や予約日がバラバラでしたり、予約期間が長いもの、そもそも発売日が大まかにしか決まっていないものと多種多様あり管理が大変でした。
しかも様々な購入サイトや店舗で発売される為、どの商品がどこから発売されるのかという点も混乱を極めます。
推し活ばかりに時間を取られるわけにはいかない。そんな中。重複購入と購入忘れをしてしまったのです。

欲しいアイテムなど絶対買い逃さないように＆複数購入しないようにしたい。種類関係なくまとめて管理したいという思いから作ろうと思いました。


## ユーザー層について
・様々なグッズなどを収集する人
・予約商品をよく買う人
・自分の好きなタイミングで購入する人
・所持品を種類関係なくまとめて管理したい人


## なぜこの層にしたか
・発売日や購入先がバラバラで、沢山ある故に購入したかどうか忘れてしまうことを防止することに役立ててほしいと考えた為。
・急な出費に備え、購入をギリギリにする方もおられるので、購入予定日を決めることが役に立つと考えた為。
・所持品の購入場所、日付、金額も記載できるのでトラブルが合っても問い合わせ先などが明確にできると考えた為。


## サービスの利用イメージ
【欲しい商品を見つけた！】
1. 商品情報を入力
2. 予約又は購入したら、ボタンを押すだけ！

【持っている商品を確認したい】
1. 一覧から確認
2. 多い時はタグで絞込


## ユーザーの獲得について
SNSでの宣伝、Mattermostでの宣伝

## サービスの差別化ポイント・推しポイント
| サービス | 特徴 | 差別化ポイント |
| ---- | ---- | ---- |
| Mono-モノ管理 | 所持しているアイテムを確認できる | 所持品だけでなく未所持の購入したいアイテムも確認できる |
| 推しノート | 欲しいグッズの発売日や支払い日が登録できる | 発売日など以外にも予約期間の管理ができる。推し活以外にも活用できる。 |
